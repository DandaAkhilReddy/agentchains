# AgentChains Roadmap

Public roadmap for AgentChains development. Updated monthly.

## Current: v0.4.0 (Stable)

- 82+ REST API endpoints (v1 + v2)
- MCP protocol server (8 tools, 5 resources)
- 4-stage trust verification pipeline
- Creator economy with USD-first billing
- Dual-layer marketplace (builders + buyers)
- Real-time WebSocket event streaming
- 2,369+ backend tests, 376+ frontend tests
- Azure Container Apps deployment

## Q1 2026: WebMCP Integration (v0.5.0)

- [ ] WebMCP tool registration and discovery
- [ ] ActionListing model (executable actions as marketplace items)
- [ ] Proof-of-execution with JWT signatures
- [ ] v3 API namespace for WebMCP endpoints
- [ ] 3 new MCP tools for WebMCP operations
- [ ] Frontend Actions tab with dynamic execution forms
- [ ] Domain Lock and Tool Lock security
- [ ] Docker sandbox isolation for action execution

## Q1 2026: A2A Protocol (v0.6.0)

- [ ] A2A server implementation (`.well-known/agent.json`)
- [ ] A2A client SDK for agent-to-agent communication
- [ ] Agent composition (chain multiple agents into pipelines)
- [ ] Pipeline marketplace listings

## Q2 2026: Community & Launch (v1.0.0)

- [ ] ProductHunt and HackerNews launch
- [ ] Demo agents (WebMCP shopping, research)
- [ ] OpenAPI/Swagger documentation
- [ ] Blog posts and tutorials
- [ ] Discord community

## Q2-Q3 2026: Production Hardening

- [ ] Real Stripe/Razorpay payment integration
- [ ] Redis-backed rate limiting for multi-instance
- [ ] PostgreSQL migration tooling (Alembic)
- [ ] OpenTelemetry observability
- [ ] Performance benchmarks and optimization
- [ ] Multi-region deployment support
